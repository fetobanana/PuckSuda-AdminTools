Commands to modify the score, timer, period, and trigger faceoff and game end.

Server side mod, no need for clients to use it.

Available commands:

'/faceoff' — Triggers a faceoff and resumes timer if paused.

'/set redscore #' or '/s rs #' — Sets the Red team's score. Example: /set redscore 3.

'/set bluescore #' or '/s bs #' — Sets the Blue team's score. Example: /s bs 2.

'/set period #' or 'set p #' — Changes the current period. Example: /set period 2.

'/set timer [mm:ss]' or '/s t [mm:ss] — Adjusts the clock. Example: /s t 2:30.

'/endgame' - Triggers a game over.

Made for Puck Sudamérica, but you can use it on your server.
